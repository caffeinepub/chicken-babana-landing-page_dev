# Chicken Babana Landing Page

## Overview
A minimal landing page that displays "Chicken Babana" as the main content, centered on the screen with clean and simple styling.

## Features
- Display "Chicken Babana" text centered on the page
- Clean white background
- Large bold black text using modern sans-serif font
- Fully responsive design that works on all screen sizes
- Minimal styling for fast loading

## Technical Requirements
- No backend functionality required - this is a static frontend-only page
- No data persistence needed
- No user interactions or dynamic content
- Optimized for quick loading with minimal CSS

## Design Specifications
- Background: White
- Text: Large, bold, black color
- Font: Modern sans-serif typeface
- Layout: Centered horizontally and vertically
- Responsive: Adapts to different screen sizes while maintaining centered layout
